DESIGN





My web page design read Google material design (https://getmdl.io/) for reference. The common theme of the web pages are:
header with blue background, a picture of a colorful lion under the header, a black lion for web page icon and the footer.
Codes for the footer are copied from material design website, the following codes are original codes from the website:
A little change I made is, when the mouse over 'Home' and 'Design' which are two links, the text size and color will change.

<footer class="mdl-mini-footer"> <div class="mdl-mini-footer__left-section"> <div class="mdl-logo">Title</div> <ul class="mdl-mini-footer__link-list"> <li><a href="#">Help</a></li> <li><a href="#">Privacy & Terms</a></li> </ul> </div> </footer>

One important thing is, I import three css files and one javascript file, for achieving the shadow effect and some animation, for example,
in index page, the animation after click the botton. I use that js file only use for the design and having a good looks of the websites.

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons"> <link rel="stylesheet" href="https://code.getmdl.io/1.3.0/material.indigo-pink.min.css"> <link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Roboto:300,400,500,700" type="text/css"> <script defer src="https:/code.getmdl.io/1.3.0/material.min.js"></script>

The class names such as 'mdl-mini-footer__left-section' and 'mdl-textfield mdl-js-textfield', are effects provided by material design.
A class which is named 'demo-content mdl-color--white mdl-shadow--4dp content mdl-color-text--grey-800 mdl-cell mdl-cell--3-col', it's
showing a block white background and shadow effect.
